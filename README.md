# divide-into-groups


# OVERVIEW
when you want to big groups into small groups, this service may be useful.
大人数をランダムな小グループ分け出来るツール

例)

("A", "B", "C", "D", "E", "F", "G", "H", "I", "J", "K", "L", "M", "N"))
